## 字符串

#### <a href="./string/huiwen.c">demo1</a>
如果参数字符串是个回文，函数就返回真，否则返回假。回文就是指一个字符串从左向右和从右向左读是一样的。函数应该忽略所有的非字母字符，而且在进行字符比较时不用区分大小写
